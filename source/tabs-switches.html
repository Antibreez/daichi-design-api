Табы

Табы группируют контент и помогают в навигации. У табов помимо названия может присутствовать счетчик для содержимого.

По умолчанию

Выбрано

При наведении

Фокус. Клавишей Tab первый таб получает фокус и появляется синяя рамка. Стрелками ← → можно перемещать фокус между табами. Если нажать клавишу Enter — произойдет переход.

Переключатель (Switcher)

Используется как замена группе радиокнопок.

По умолчанию

Выбрано

При наведении

Неактивно

Ошибка

Фокус. Клавишей Tab пункт получает фокус и появляется синяя рамка. Стрелками ← → можно перемещать фокус между пунктами. Если нажать клавишу Enter — произойдет выбор.

Переключатель c текстом

Работает как и стандартный переключатель, но содержит текстовый блок. Есть несколько вариантов: стандартный с текстом, текст с иконками, только иконки.
