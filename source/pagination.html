Пагинация

Постраничная навигация используется для представления большого количества однородной информации. Для удобства перехода по страницам используем пагинацию в связке с кнопкой «Показать еще».

При наведении

Фокус. Клавишей Tab первая страница получает фокус и появляется синяя рамка. Стрелками ← → можно перемещать фокус между страницами. Если нажать клавишу Enter — произойдет переход.
